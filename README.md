# chat.aswincloud.com

A standalone full-viewport chat page that iframes the self-hosted Chatwoot
widget at `support.aswincloud.com`. Lets visitors land on
`chat.aswincloud.com` and start a conversation immediately, without any
portfolio chrome.

## Deployment

Deployed via Cloudflare Pages. Pushes to `main` auto-deploy.

The site is a single static `index.html` — no build step.
